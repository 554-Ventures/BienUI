## Quick Reference - Brand Colors

**Primary Brand Colors:**
- Blue: #1d75bc (Primary buttons, links, key UI)
- Blue Dark: #155a94 (Hover states)
- Teal: #016d77 (Secondary brand accent)

**Accent Colors:**
- Gold: #e7c067 (Highlights, badges)
- Purple: #a94f9e (Creative features)
- Purple Dark: #8b3f85 (Purple hover states)

**Status Indicators:**
- Success: #14b8a6 (Confirmations, positive actions)
- Warning: #fbbf24 (Caution messages)
- Error: #d4183d (Errors, destructive actions)
- Info: #3b82f6 (Information messages)

## Usage Guidelines

1. **Primary Colors** should be used for main brand elements, primary actions, and navigation
2. **Accent Colors** add visual variety and can highlight special features or content
3. **Gradients** create depth and modern visual appeal for cards, buttons, and hero sections
4. **Status Colors** provide clear feedback and should be used consistently for their respective states
5. **Text Colors** establish hierarchy - use primary for headings, secondary for body text, tertiary for metadata
6. **Background Colors** create layered layouts with subtle depth
7. **Border Colors** define elements with appropriate emphasis levels
